Formulários Modernos

Os formulários HTML5 oferecem novos tipos de input, validação nativa e melhor experiência do usuário. Aprenda a criar formulários interativos e acessíveis.

Novos Tipos de Input

type="email"

Validação automática de email

type="tel"

Campo para números de telefone

type="date"

Seletor de data nativo

type="range"

Controle deslizante

type="color"

Seletor de cores

type="number"

Apenas números

Validação HTML5

<form> <label for="email">Email:</label> <input type="email" id="email" required> <label for="idade">Idade:</label> <input type="number" id="idade" min="18" max="100" required> <label for="site">Website:</label> <input type="url" id="site" pattern="https://.*"> <button type="submit">Enviar</button> </form>

Formulário Interativo

5

Atributos de Validação

  • required: Campo obrigatório
  • minlength/maxlength: Tamanho mínimo e máximo
  • pattern: Expressão regular para validação
  • min/max: Valores mínimo e máximo para números
