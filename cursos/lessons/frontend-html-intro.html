Introdução ao HTML5

Módulo 1 • Aula 1 • 15 minutos

O que é HTML5?

HTML5 é a quinta versão da linguagem de marcação HTML, que define a estrutura e o conteúdo de páginas web. É a base fundamental de qualquer site moderno.

Estrutura Básica do HTML5

<!DOCTYPE html> <html lang="pt-BR"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Minha Primeira Página</title> </head> <body> <h1>Olá, Mundo!</h1> <p>Esta é minha primeira página HTML5.</p> </body> </html>

Principais Novidades do HTML5

  • Elementos Semânticos: <header>, <nav>, <main>, <section>, <article>, <footer>
  • Multimídia: <video>, <audio>, <canvas>
  • Formulários: Novos tipos de input como email, date, range
  • APIs: Geolocalização, Local Storage, Web Workers
